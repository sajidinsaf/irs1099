# Getting Started

Welcome to the **IRS 1099 Filing Platform** -- the easiest way to file your IRS 1099 information returns electronically.

## What is this platform?

Our platform lets you file 1099 forms directly with the IRS through their IRIS (Information Returns Intake System) A2A channel. Instead of dealing with complex XML files and IRS technical specifications, you simply:

1. **Create an account** and set up your business profile
2. **Enter your 1099 data** through guided forms
3. **Pay** per form or subscribe to a plan
4. **Submit** -- we handle the rest and send your forms directly to the IRS
5. **Track** your filing status in real-time

## Who needs to file 1099s electronically?

If you have **10 or more information returns** to file in a calendar year, you are **required to file electronically** per IRS regulations. Even if you have fewer than 10, electronic filing is encouraged.

## Security

Your data is protected with:
- **AES-256 encryption** for all TIN/SSN/EIN data
- **BCrypt password hashing**
- **HTTPS/TLS** for all communications
- Full compliance with **IRS Publication 4557** (Safeguarding Taxpayer Data)

We never store your TIN or SSN in plain text. Even our own staff cannot see your sensitive data.
