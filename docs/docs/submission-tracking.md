# Tracking Your Submissions

After submitting your 1099 forms to the IRS, you can track the status in real-time.

## Submission statuses

| Status | Description | What to do |
|--------|-------------|------------|
| **Draft** | Not yet submitted | Add records and submit |
| **Validating** | Being validated locally | Wait -- this is automatic |
| **Submitted** | Sent to the IRS | Wait for IRS processing |
| **Processing** | IRS is reviewing | Check status periodically |
| **Accepted** | All records accepted | Nothing -- you're done! |
| **Accepted with Errors** | Some records had issues | Review errors, file corrections |
| **Rejected** | Submission was rejected | Fix issues and resubmit |

## Checking status

1. Go to your dashboard
2. Click on a submission to view details
3. Click **Check Status** to get the latest from the IRS

:::tip
The IRS typically processes submissions within a few minutes, but it can take up to 48 hours during peak filing periods.
:::

## Receipt ID

When your submission is accepted by the IRS, you receive a **Receipt ID**. This is your confirmation number. Keep it safe -- you'll need it for:
- Corrections
- Replacements
- Contacting the IRS about your filing

## Error handling

If your submission is rejected, the error details will be displayed on the submission page. Common errors include:
- Invalid TIN format
- Mismatched TIN and name
- Missing required fields
- Duplicate transmission ID

Fix the issues and create a new submission, or file a correction if the original was partially accepted.

## Filing history

Your dashboard shows all submissions with:
- Form type and tax year
- Number of records
- Current status
- Submission date
- Receipt ID (when available)
